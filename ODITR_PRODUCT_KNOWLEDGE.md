# Øditr - Product Knowledge Base

## 1. Product Identity
Øditr (pronounced "Auditor") is a developer-first web intelligence and website audit platform. It's designed to help developers, founders, agencies, and product teams find what is slowing, breaking, blocking, or hiding their website — across Lighthouse, Core Web Vitals, SEO, accessibility, security, and AI-agent readiness.

It goes beyond standard SEO checkers or Lighthouse score generators. It is a web intelligence engine that tells users what to fix, why it matters, and the potential business impact.

## 2. Core Product Pillars
1. **Lighthouse + Core Web Vitals**: Deep performance metrics including LCP, CLS, INP, page weight, JS/CSS size.
2. **Technical SEO**: Focus on metadata, schema, crawlability, and indexing logic.
3. **Accessibility**: Color contrast, ARIA, keyboard navigation, and WCAG severity.
4. **Security & Protocol Auditing**: Headers, CSP, HSTS, TLS, and mixed content.
5. **AI-Agent Readiness**: A novel score checking for llms.txt, robots.txt for AI crawlers, semantic HTML, and accessibility tree quality.
6. **Developer Fix Engine**: Exact evidence, recommended fixes, and framework-specific code snippets.
7. **Monitoring & Regression Detection**: CI/CD integration, scheduled audits, historical charts, and alert systems.

## 3. Market Direction
Øditr focuses on the technical website health layer, differentiating itself from pure marketing SEO tools by offering deep code-level debugging, real-time inspection, and developer-first workflows with exact implementation fixes.

## 4. UI/UX Direction
